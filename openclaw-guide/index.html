安装步骤

按顺序执行，遇到问题看下方常见问题

1

安装 Homebrew

macOS 的软件包管理器，已有可跳过

打开"终端"（Terminal）App，复制下面这行命令粘贴进去，按回车。过程中可能需要输入你的 Mac 登录密码。

Terminal
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
💡
Apple Silicon（M1/M2/M3/M4）Mac 用户注意： 安装完成后，还需要运行下面这行命令让 Homebrew 生效：
eval "$(/opt/homebrew/bin/brew shellenv)"
✅
验证方法 输入 brew --version，显示版本号（如 Homebrew 4.x.x）即表示成功
2

安装 Node.js

运行 OpenClaw 所需的 JavaScript 运行环境

在终端中运行：

Terminal
brew install node

安装过程可能需要几分钟，等待完成即可。

✅
验证方法 运行 node -v，显示版本号（如 v22.x.x）即表示成功
3

安装 OpenClaw

小金助手的核心程序

在终端中运行：

Terminal
npm install -g openclaw
✅
验证方法 运行 openclaw --version，显示版本号即表示安装成功
4

启动 OpenClaw

按照引导完成初始配置

在终端中运行：

Terminal
openclaw start

程序会引导你完成以下配置：

  • 选择 AI 提供商：Claude 或 OpenAI
  • 输入你的 API Key
  • 其余选项可以直接回车使用默认值
✅
验证方法 打开浏览器访问 http://localhost:3000，能看到界面即表示成功启动！
5

绑定 Telegram

可选但强烈推荐，随时随地用手机聊天

绑定 Telegram 后，你可以用手机直接和小金助手聊天，非常方便。

  1. 打开 Telegram，在搜索框搜索 @BotFather 并点击进入
  2. 发送 /newbot，按提示给你的 Bot 起个名字（比如"我的小金"）和用户名
  3. BotFather 会给你一串 Bot Token，格式类似 1234567890:ABCDEF...，复制保存好
  4. 回到 OpenClaw 配置界面，在 Telegram 插件设置中填入这个 Token
  5. 配置保存后，在 Telegram 搜索你刚创建的 Bot，点击开始聊天 🎉
✅
验证方法 在 Telegram 中给你的 Bot 发一条消息，收到回复即表示配置成功！

常见问题

遇到问题先看这里

Q 提示 npm: command not found 怎么办？
说明 Node.js 没有安装成功，请重新执行第二步：brew install node。如果 brew 也找不到，则需要先重新执行第一步安装 Homebrew。
Q API Key 在哪里申请？
· Claude API Key：访问 console.anthropic.com，注册账号后在 API Keys 页面创建
· OpenAI API Key：访问 platform.openai.com/api-keys，登录后创建新 Key

申请需要信用卡，建议先充值少量金额（$5 即可）开始体验。
Q Gateway 是什么？
Gateway 是 OpenClaw 的核心后台服务，负责接收来自 Telegram / 浏览器等渠道的消息，并调用 AI 生成回复。运行 openclaw start 时 Gateway 会自动启动。你也可以用 openclaw gateway status 查看运行状态。
Q Apple Silicon Mac 上 brew 命令找不到？
M 系列芯片的 Mac，Homebrew 安装在 /opt/homebrew/ 目录下，需要手动让终端识别。在终端运行：
eval "$(/opt/homebrew/bin/brew shellenv)"

要永久生效，可以将这行加入 ~/.zprofile 文件中。
Q 如何关闭 / 重启 OpenClaw？
· 查看状态：openclaw gateway status
· 停止服务：openclaw gateway stop
· 重启服务：openclaw gateway restart
