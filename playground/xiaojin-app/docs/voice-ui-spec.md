# 小金语音 UI 行为规范

## 场景一：初始状态
- Given App 刚打开
- Then 显示白色背景
- Then 中间显示黑色球体，缓慢呼吸动画
- Then 底部显示3个按钮：拍照/上传、麦克风、关闭
- Then 右上角显示 CC 按钮

## 场景二：连接 Gateway
- Given App 开始连接 Gateway
- Then 球体从黑色渐变为蓝白渐变色（1.5秒过渡）
- Then 连接成功后球体轻微上下浮动

## 场景三：用户说话
- Given 麦克风已开启
- When VAD 检测到声音
- Then 球体出现声波扩散动画
- Then 底部显示"正在听..."

## 场景四：AI 思考
- Given 用户说话结束，消息已发送
- Then 球体收缩并有旋转感
- Then 底部显示"小金正在思考..."

## 场景五：AI 说话
- Given AI 开始 TTS 播放
- Then 球体节奏性扩张动画
- Then CC 开启时，AI 回复文字显示在球下方大字
- Then CC 关闭时，文字隐藏

## 场景六：麦克风开关
- Given 用户点击麦克风按钮
- When 麦克风当前开启
- Then 麦克风关闭，按钮变红色+斜线图标
- When 麦克风当前关闭
- Then 麦克风开启，按钮恢复正常

## 场景七：拍照/上传
- Given 用户点击拍照/上传按钮
- Then 弹出选项：「上传照片」「拍照」
- When 用户选择「上传照片」
- Then 调用系统图片选择器
- When 用户选择「拍照」
- Then 调用系统相机

## 场景八：关闭对话
- Given 用户点击关闭按钮
- Then 断开 Gateway 连接
- Then 球体回到黑色呼吸状态
